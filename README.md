
# Mondrian Tetris

A simple Tetris game implemented in Python using the Pygame library. This version features a "Petite Mondrian" color scheme for a unique visual style.

## Features

*   Classic Tetris gameplay
*   "Petite Mondrian" color palette (red, blue, yellow, white, and black)
*   Standard 10x20 Tetris playing field
*   Hard drop feature (spacebar)
*   Rotation of tetrominoes

## Requirements

*   Python 3
*   Pygame

## Installation & Usage

1.  **Clone the repository:**

    ```bash
    git clone <repository-url>
    cd tetris_mondrian
    ```

2.  **Install Pygame:**

    ```bash
    pip3 install pygame
    ```

3.  **Run the game:**

    ```bash
    python3 main.py
    ```

## Controls

*   **Left Arrow:** Move the tetromino left
*   **Right Arrow:** Move the tetromino right
*   **Down Arrow:** Move the tetromino down
*   **Up Arrow:** Rotate the tetromino
*   **Spacebar:** Hard drop the tetromino to the bottom
